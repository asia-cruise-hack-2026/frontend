# 정찰 노트 ③ — 제주관광공사 (ijto.or.kr)

- **정찰일**: 2026-07-21 (chrome-devtools MCP)
- **성격**: 제주관광공사(JTO) **기관 공식 사이트**. 서버 렌더링 PHP. 정책·기관·홍보 중심.
- **핵심 발견**: **관광 통계·연구자료실은 빅데이터플랫폼(data.ijto.or.kr)으로 이관 완료**됨. 따라서 통계 데이터는 [정찰 노트 ①] 참조. 이 사이트의 고유 가치는 **공식 홍보/정책/기관 정보 + 공공데이터 신청 창구**.

## 1. 활용 가능한 콘텐츠

| 항목 | URL | 성격 |
|---|---|---|
| **보도자료** | `/korean/Bd/list.php?btable=report_info` | 공사 사업·캠페인·행사 공식 발표. **살아있는 트렌드/정책 신호** |
| 공공데이터 제공(신청) | `/korean/index.php?cid=138` | 공공데이터법 제27조 **신청 창구**(양식). 즉시 데이터 아님 |
| 공공데이터 신청·활용제안 | `/korean/index.php?cid=246` | 활용 제안 접수 |
| 사전정보공표/경영공시 | `/korean/beforeList/`, 정보공개 메뉴 | 예산·사업계획·조직 등 거버넌스 |
| 관광자료실(이관) | → `https://data.ijto.or.kr/` | 통계/보고서는 빅데이터플랫폼으로 이동 |

- **게시판 패턴**: `Bd/list.php?btable={board}&p={page}` (목록), `Bd/view.php?btable={board}&bno={번호}` (상세), 검색 지원.
  - `report_info` = 보도자료, `pds` = 자료실/공지 등.

## 2. 실데이터 스냅샷 — 보도자료 최신 (2026-07, 캡처 2026-07-21)
공사가 현재 밀고 있는 사업·이벤트(=이데이션 정책 맥락):
- 이호 해변 여름 축제 (수변/해변 축제)
- **제주관광 현장에 AI 입힌다 — 관광기업 찾아가는 맞춤형 교육**
- 여름휴가 반값 — 제주 촌(村)플레이 기획전
- **반려동물과 함께하는 여행 확대 — ᄒᆞᆫ저옵서개 참여업체 모집**
- 제주 수변공원 ESG 축제 첫 개최
- 안전 인증 농어촌민박 한 달 살기 (50팀 모집에 178팀)
- **제9회 제주 MICE 산업대전 / 제주 MICE 앰배서더**
- 원도심 상권조합과 민·관 협력
- **외국인 관광객 20% 할인 — 제주관광 쇼핑 페스타**

> 키워드: AI·MICE·ESG·반려동물·원도심·외국인 쇼핑·한 달 살기 — 공사의 전략 방향성이 드러남.

## 3. 관련(유관) 기관 — 2차 데이터 소스 후보
푸터 유관기관 링크: 제주연구원, **제주특별자치도관광협회**(빅데이터 일부 출처), 제주통계사무소, (사)제주컨벤션뷰로, (주)제주국제컨벤션센터, 한국방문위원회, 제주테크노파크 등.
→ 크루즈/MICE/통계 심화 시 추가 정찰 대상(이번 범위 밖, 후보로 기록).

## 4. 활용 각도 (이데이션 참조)
- **정책·전략 맥락**: 보도자료로 공사가 현재 추진하는 테마(AI 관광, MICE, ESG, 반려동물, 외국인 쇼핑) 파악 → 기획을 기관 방향성과 정렬.
- **행사/캠페인 연계**: 진행 중 이벤트를 아이디어의 파트너십·시즌 훅으로 활용.
- **데이터 확보 경로**: 대량·안정 데이터가 필요하면 '공공데이터 신청'(cid=138) 정식 경로 확보 — 익명 스크래핑의 백업.
- **크루즈/MICE**: MICE 산업대전·앰배서더, 크루즈 관련 보도로 B2B/유치 맥락 확인.
